# Website Static Lead Privacy Note

## Current Handling

- the current website is static
- territory submissions are captured manually from browser storage or recorded manually
- no backend collection service is introduced in this sweep

## Minimum Data-Handling Rules

- collect only the documented territory fields
- keep captured lead data in internal operational storage only
- do not publish captured lead files with the static site
- limit access to people handling website lead review and ops follow-up

## Retention Note

- keep lead records only as long as needed for the current review and follow-up workflow
- if a captured lead is no longer active, remove it from the working intake store according to internal handling practice

## Scope Note

- this note is operational guidance for the current static workflow only
- it does not add or change website runtime behavior
