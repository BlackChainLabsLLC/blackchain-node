# Founder First Look Every Day

## Look Here First

1. Check the founder daily dashboard for top priority, blocker, and decision needed.
2. Check new website leads and follow-ups due today.
3. Check territory pipeline movement and any qualified leads waiting for action.
4. Check active launches, activations in progress, and open launch blockers.
5. Check the founder decision queue for overdue decisions.

## If Something Is Off

- if lead response is behind, review the first-response workflow and templates
- if territory progression is behind, review the territory pipeline handoff and tracker
- if launch movement is behind, review the launch summary and current blockers

## Goal

- see the lead, territory, and launch picture in one pass before starting new work
