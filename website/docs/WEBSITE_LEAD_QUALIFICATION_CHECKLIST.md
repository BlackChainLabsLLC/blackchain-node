# Website Lead Qualification Checklist

## Goal

- move a website lead from `new` to `qualified` using the current ops review path

## Review Checks

- contact details are usable
- territory or town is clearly identified
- host-site interest is clear
- operator interest is clear
- preferred plan tier is captured or intentionally blank
- review notes have been added
- next action and owner are assigned

## Qualification Decision

- mark `qualified` only when the lead can move into active territory review
- if the lead needs more information first, keep it in `contacted` or `reviewing`
- update the tracker immediately after the decision
