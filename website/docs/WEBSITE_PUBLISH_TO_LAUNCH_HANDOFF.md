# Website Publish To Launch Handoff

## Goal

- connect website publish readiness to launch-path operations without changing deployment behavior

## Handoff Notes

- website publish confirms the public pages, CTA routes, and territory capture flow are live
- launch ops still owns territory review, host-site follow-up, operator follow-up, and activation review
- website publish does not replace ops review or launch qualification

## Confirm Before Handoff

- static publish is live on the intended host
- all current routes load
- territory submit reaches `/territory/thanks/`
- capture workflow reference is available to the reviewer
- lead tracker template is ready for use

## Handoff Output

- published website URL
- publish timestamp
- reviewer owner
- lead capture workflow reference
- rollback reference
