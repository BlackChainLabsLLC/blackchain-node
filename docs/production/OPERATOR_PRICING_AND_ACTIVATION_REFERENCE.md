# BlackChain Operator Pricing And Activation Reference

This reference is for operator launch handling only.

## Plan Tiers

| Plan Tier | Hardware Issuance | Activation Notes |
| --- | --- | --- |
| Standard | Pi Zero | baseline activation path |
| Premium | Pi 3 | elevated customer support expectation |
| Family/Business | Pi 4 | higher-capacity host-site expectation |

## Operator Activation Fields

- customer or site name
- selected plan tier
- issued hardware serial or asset tag
- host-site onboarding sheet completed
- customer activation checklist completed
- launch owner assigned
- support owner assigned

## Activation Rule

- no activation should proceed without plan tier, issued hardware, and release identifier recorded together
