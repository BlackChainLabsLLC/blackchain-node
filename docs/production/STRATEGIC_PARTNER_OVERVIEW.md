# BlackChain Strategic Partner Overview

## Partnership Thesis

BlackChain uses a host-site and operator-supported node launch path that can be repeated across territories without changing the core deployment shape.

## What Partners Engage With

- host-site onboarding
- customer activation coordination
- operator and support assignment
- launch and monitoring process

## Why The Current Path Is Partner-Friendly

- fixed host paths and deployment shape
- defined launch checklist and review structure
- defined customer activation and support templates
- hardware issuance aligned to commercial tiers
