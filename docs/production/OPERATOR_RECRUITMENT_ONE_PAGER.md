# BlackChain Operator Recruitment One-Pager

## Operator Role Summary

Operators handle deployment, activation, verification, monitoring, and issue capture for the current BlackChain launch path.

## Operator Responsibilities

- place binaries, configs, and certificates
- enable and monitor the current node set
- run smoke and field verification
- support customer activation and post-launch checks
- maintain daily ops logs and incident records

## Operator Tooling

- release and install bundle
- field verification and smoke scripts
- Town v1 handoff pack
- pilot operations pack
- commercial launch pack
