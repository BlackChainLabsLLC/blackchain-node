# BlackChain Founder Talking Points

## Core Story

- BlackChain is a node-based launch path built for repeatable town-by-town rollout.
- The current path uses a fixed deployment shape, named operators, host-site coordination, and customer activation tied to plan tiers.
- The model is structured, documentation-first, and operationally defined.

## Commercial Story

- Standard maps to Pi Zero.
- Premium maps to Pi 3.
- Family/Business maps to Pi 4.
- Activation is coordinated, verified, and supported through named operational roles.

## Why Now

- the launch path is defined
- operator workflows are documented
- pilot proof and launch materials are in place
- territory and partner materials are ready for use

## Founder Close

- the current goal is disciplined rollout, not protocol redesign
- the value is repeatable activation, support, and territory expansion
