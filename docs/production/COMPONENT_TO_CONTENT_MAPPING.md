# BlackChain Component To Content Mapping

## Homepage

- `HeroSection` -> homepage hero
- `IntroSection` -> homepage intro
- `HowItWorksSection` -> homepage how-it-works
- `AudienceSection` -> homepage audience
- `PrimaryCtaGroup` -> homepage CTAs

## Pricing Page

- `PricingOverview` -> pricing overview
- `PlanTierCard` -> Standard, Premium, Family/Business tiers
- `ActivationNote` -> pricing activation note

## Host-Site Page

- `LandingHeader` -> host-site header
- `ValueSection` -> host-site value
- `ResponsibilityList` -> host-site responsibilities
- `ExpectationList` -> host-site expectations
- `PrimaryCtaGroup` -> host-site CTAs

## Operator Page

- `LandingHeader` -> operator header
- `ValueSection` -> operator value
- `ResponsibilityList` -> operator responsibilities
- `FitSection` -> operator fit
- `PrimaryCtaGroup` -> operator CTAs

## Territory Page

- `LandingHeader` -> territory title
- `IntroSection` -> territory intro
- `TerritoryInterestForm` -> territory form
- `PrimaryCtaButton` -> territory submit CTA
