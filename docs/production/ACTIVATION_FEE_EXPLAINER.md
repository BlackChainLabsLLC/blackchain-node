# BlackChain Activation Fee Explainer

This explainer is for operator and sales alignment during launch planning.

## Activation Fee Purpose

- covers launch coordination
- covers hardware issuance and activation handling
- covers operator setup and verification effort

## Activation Inputs

- selected plan tier
- issued hardware tier
- host-site onboarding status
- customer activation status
- release identifier

## Operator Reminder

- activation should not proceed without plan tier, hardware issuance, and customer activation records aligned
