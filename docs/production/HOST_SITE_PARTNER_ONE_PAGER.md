# BlackChain Host-Site Partner One-Pager

## What A Host-Site Partner Provides

- placement location for the current BlackChain node deployment shape
- local coordination for site onboarding and activation
- access coordination for operator verification and support

## Current Deployment Shape

- binaries under `/usr/local/bin`
- configs under `/etc/blackchain`
- data under `/var/lib/blackchain`
- systemd unit at `/etc/systemd/system/blacknetd@.service`

## What Partners Should Expect

- a defined activation window
- a host-site onboarding sheet
- named operator and support contacts
- a verification and monitoring process after launch
