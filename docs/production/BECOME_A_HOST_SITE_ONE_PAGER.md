# Become A Host Site

Host sites help make local BlackChain rollout possible.

## What A Host Site Does

- provides a launch location for the current node deployment shape
- coordinates local access and onboarding
- supports activation timing and operator verification

## What To Expect

- a defined launch window
- a site onboarding process
- named operator and support contacts
- post-launch monitoring and support follow-up

## Good Fit

- willing local coordination
- reliable site access
- readiness for onboarding and scheduled activation
