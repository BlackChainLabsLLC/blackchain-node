# BlackChain Website Design And Content System Notes

## Content Rules

- keep copy short and direct
- use the existing plan tier names exactly
- keep hardware mapping consistent:
  - Standard -> Pi Zero
  - Premium -> Pi 3
  - Family/Business -> Pi 4

## Page Rules

- one primary CTA per section
- keep landing pages aligned to the current launch path
- do not introduce protocol or product claims beyond the current docs

## Tone

- public-facing
- practical
- rollout-oriented
- easy to scan
