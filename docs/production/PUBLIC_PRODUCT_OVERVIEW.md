# BlackChain Public Product Overview

BlackChain is a node-based product launch path built for town-by-town rollout.

## What BlackChain Offers

- a structured launch path for local deployment
- customer activation tied to simple plan tiers
- operator and host-site support for onboarding and rollout

## Current Launch Path

- Standard plan maps to Pi Zero
- Premium plan maps to Pi 3
- Family/Business plan maps to Pi 4

## How Launch Works

- a host site is prepared
- an operator provisions and verifies the deployment
- the customer is activated and supported through launch
