# Become An Operator

Operators help deploy, activate, verify, and monitor BlackChain in the field.

## What Operators Do

- place and verify the launch package
- activate the current node deployment shape
- run smoke and field verification checks
- support customer onboarding and issue capture

## What Operators Use

- provisioning and launch checklists
- field verification and smoke scripts
- support and escalation templates
- launch and pilot operations materials

## Good Fit

- technically reliable
- comfortable with operational checklists
- able to support activation and early monitoring
