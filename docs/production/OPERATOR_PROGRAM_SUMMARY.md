# BlackChain Operator Program Summary

This summary describes the current operator program for the BlackChain launch path.

## Operator Scope

- deploy binaries and configs to the current host layout
- place CA and node certificates
- enable and operate the current node set
- run smoke, verification, and monitoring checks
- capture support and incident artifacts

## Operator Tooling

- release manifest template
- promotion and rollback scripts
- provisioning helper
- post-install smoke
- field verification
- Town v1 handoff pack
- pilot operations pack

## Commercial Mapping

- Standard customers map to Pi Zero issuance
- Premium customers map to Pi 3 issuance
- Family/Business customers map to Pi 4 issuance

## Success Conditions

- activation completes with documented signoff
- HTTPS and `blackctl` trust are verified
- monitoring ownership is assigned
- rollback remains immediately available
