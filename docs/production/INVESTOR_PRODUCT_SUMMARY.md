# BlackChain Investor Product Summary

## Product Summary

BlackChain is a node-based launch path built around a fixed deployment shape, operator-managed activation, and tiered hardware issuance.

## Current Launch Model

- fixed deployment topology for the current launch path
- operator-led installation, verification, and monitoring
- customer activation tied to plan tier and issued hardware

## Commercial Mapping

- Standard -> Pi Zero
- Premium -> Pi 3
- Family/Business -> Pi 4

## Operating Model

- release manifest and deployment assets
- Town v1 operator handoff pack
- pilot operations pack
- commercial launch pack

## Investor Notes

- launch path is documentation-first and operationally structured
- operator and support workflows are defined
- rollout readiness is evaluated through launch and scorecard materials
